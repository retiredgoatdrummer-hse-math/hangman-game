Игра "Виселица"

Консольная игра с двумя режимами получения слов: из локальной базы или через API Merriam-Webster.

I. Правила

1. Программа случайным образом выбирает слово и отображает его в виде звездочек ( * )
2. Игрок вводит по одной букве английского алфавита
3. Если буква есть в слове, она открывается на соответствующих позициях
4. Если буквы нет в слове, счетчик попыток уменьшается на 1
5. Игра продолжается пока:
   - Игрок не откроет все буквы (победа)
   - Не закончатся попытки (поражение)
6. В конце игры показывается определение слова

II. Функциональность

1. Режимы игры
- Режим 1: Использование предустановленной базы 'base_installed.json'
- Режим 2: Генерация слов через Merriam-Webster API

2. Игровой процесс
- Отображение замаскированного слова символами " * "
- Ввод букв с проверкой на корректность:
  - Только английские буквы
  - Только один символ
  - Отсутствие повторов
- Счетчик попыток = длина слова + дополнительные попытки
- Отображение угаданных букв
- Показ определения слова в конце игры
- История угаданных слов

III. Структура файлов
- 'main_game.ipynb' — основной игровой ноутбук
- 'base_installed.json' — база слов и определений
- 'base_construction.ipynb' — файл для сбора базы (пользователь может самостоятельно сгенерировать новую)

IV. Источники данных
- Локальная база: предварительно собранные слова (5000+)
- Онлайн-режим: Merriam-Webster Dictionary API
- Генерация слов: NLTK WordNet (существительные длиной больше 6 букв)

V. Технические требования
- Python 3.7+
- Библиотеки: 'nltk', 'requests', 'json', 'secrets' (для создания базы также 'os', 'time')
